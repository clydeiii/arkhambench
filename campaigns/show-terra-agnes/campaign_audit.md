## Finding 1 — The Devourer Below offers a blanket "Resign" action that no card in the scenario grants

**Step:** c-show-terra-agnes-3 (return_to_the_devourer_below), R1–R2 action menus; taken R2 (`log.md` line ~110, "Decision made: Resign").

**Rules basis:** Rules Reference, "Resign": *"Some abilities are identified with a Resign action designator. Such abilities are initiated using the 'Activate' action."* Resigning is not a basic action; it exists only where a card provides it. In Midnight Masks that provision is the agenda text (and the engine correctly gates it: Resign appears 62 times in scenario 2's menus, **zero** times in The Gathering). No Devourer Below act or agenda carries a Resign designator — verified via card lookup on 01143–01148 (The Arkham Woods, The Ritual Begins, Vengeance Awaits, Investigating the Trail, Into the Darkness, Disrupting the Ritual) — yet scenario 3's timeline shows "Resign" in all 11 action menus, and sibling Devourer runs (e.g. show-sol-skids-3, 20 offers) show the same, so it is engine-wide, not campaign-specific.

**Outcome relevance:** Agnes resigned on round 2 at 1 damage/1 horror, converting a live game into no_resolution → Arkham succumbed → investigator killed → campaign over with score 0 for the scenario. By RAW she could not have quit; she would have been forced to keep playing with real winning chances. In show-sol-skids the agent likewise used it as an escape valve ("Resign because five engaged cultists make... immediately lethal").

**Caveats for adjudication:** (a) the run's own `mission.md` line 18 says *"resigning is no escape and records Arkham's destruction"* — so this is a documented, intentional concede affordance, not an accidental leak of MM's resign machinery; (b) I could not inspect the Return to the Devourer Below card pool (data/cards is off-limits to this audit), so if some Return card grants Resign this downgrades to NOT A BUG — but the offer being unconditional from round 1 in every DB game, on the core act/agenda set, argues against a card source. Recommend either an explicit adjudication blessing the concede affordance (and a ledger entry so future audits stop tripping on it) or gating Resign on an actual Resign ability.
